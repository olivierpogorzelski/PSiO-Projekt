# Crimson Crypt (ProjektPSiO)

Projekt gry pseudotrójwymiarowej (2.5D) stworzonej w języku C++ z użyciem techniki **Raycastingu** oraz biblioteki multimedialnej **SFML**. Projekt powstał w ramach przedmiotu *Programowanie Strukturalne i Obiektowe* (PSiO).

---

## 🎮 O Projekcie
Gra jest nowoczesną reinterpretacją klasyków w postaci *Wolfenstein 3D* i *Doom*. Gracz przemierza mroczne labirynty, zbiera przedmioty, walczy z różnorodnymi przeciwnikami i dąży do pokonania głównego bossa (Wielkiego Demona) w celu odniesienia zwycięstwa.

## ⌨️ Sterowanie

| Klawisz | Akcja |
|---|---|
| **W** / **⬆️** | Ruch do przodu |
| **S** / **⬇️** | Ruch do tyłu |
| **A** / **⬅️** | Obrót kamery w lewo |
| **D** / **➡️** | Obrót kamery w prawo |
| **Space** | Atak (strzał z aktualnej broni / uderzenie) |
| **Q** | Użycie eliksiru leczenia (potki) |
| **1** - **5** | Wybór aktywnej broni z ekwipunku |
| **Tab** | Wyświetlenie Tablicy Wyników (Scoreboard) |
| **F5** | Szybki Zapis (Quick Save) |
| **F9** | Szybki Odczyt (Quick Load) |
| **Esc** | Pauza (wywołanie menu pauzy z opcją zapisu/wczytania, powrotu do menu głównego i wyjścia) |
